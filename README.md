# amazon_send_a_question
Program to automatically post questions after login

### 【環境】
- python 3.7
- requirements.txt　参照


### 【実行前確認】
1.emailアドレスの入力
2.パスワードの入力
3.質問を投げかけたい店舗のURLを入力
4.質問の投稿をしたい文字列を入力

### 【プログラム実行後の動き】
1. web立ち上げ
2. ログイン画面
3. emailアドレス入力
4. パスワード入力
5. トップ画面
6. 【実行前確認】で入力した店舗URLへリダイレクト
7. 店舗の質問をするをクリック

8. 「問い合わせ内容を選択してください」から登録情報を選択し、メッセージを入力をクリック
